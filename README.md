🎬 Movie Recommendation System

Task 4 – KodBud Internship

📌 Project Overview

The Movie Recommendation System is a machine learning project that recommends movies to users based on
similarities between movies. It uses content-based filtering to analyze movie features such as genres,
keywords, cast, crew, and overview, and suggests similar movies.

🎯 Objective

- Build a movie recommendation system using Python.
- Recommend similar movies based on user input.
- Learn data preprocessing, feature engineering, and similarity algorithms.

🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- NLTK (optional)
- Pickle (for saving the model)

📂 Dataset

The dataset contains movie information such as:

- Movie Title
- Genres
- Keywords
- Cast
- Crew
- Overview

📈 Workflow

1. Import the dataset.
2. Clean and preprocess the data.
3. Combine important features.
4. Convert text into numerical vectors using CountVectorizer.
5. Calculate cosine similarity between movies.
6. Recommend the top similar movies based on user input.

▶️ How to Run

1. Open the Jupyter Notebook.
2. Install the required libraries.
3. Run all notebook cells.
4. Enter a movie name.
5. View the recommended movies.

📊 Output

The system displays a list of movies that are most similar to the selected movie based on content similarity.

📚 Learning Outcomes

- Data preprocessing
- Feature engineering
- Natural Language Processing (NLP)
- Cosine similarity
- Content-based recommendation systems
- Machine Learning fundamentals

✅ Conclusion

This project demonstrates how machine learning techniques can be used to build a content-based movie recommendation system that provides personalized movie suggestions efficiently.# Task4
